## 🏗️ 技术架构

### 项目结构
```
cat-book-server/
├── public/                 # 静态资源和HTML入口
│   └── index.html         # 主页面文件
├── src/                   # 源代码目录
│   ├── css/              # 样式文件
│   │   └── main.css      # 主样式文件
│   ├── js/               # JavaScript源码
│   │   ├── config.js     # 应用配置
│   │   ├── api.js        # API服务层
│   │   ├── utils.js      # 工具函数库
│   │   ├── ui.js         # UI管理器
│   │   └── main.js       # 主应用入口
│   └── api/              # API相关
│       └── models.js     # 数据模型定义
├── docs/                 # 文档目录
│   ├── api-spec.yaml     # OpenAPI 3.0.3 规范
│   ├── README.md         # API使用说明
│   └── deployment-guide.md # 部署指南
├── scripts/              # 构建脚本
│   ├── build.js          # 生产构建
│   └── validate.js       # 项目验证
├── test-demo.html        # 功能演示页面
└── 配置文件 (.eslintrc.js, .prettierrc, .gitignore 等)
```

## 🎯 核心功能

### 1. 前端功能模块
- **首页展示**: 欢迎页面、统计数据、预约须知

- **响应式UI**: 完美适配移动端和桌面端

### 2. API接口设计

- **ApiResponse Model**: 统一的响应格式
