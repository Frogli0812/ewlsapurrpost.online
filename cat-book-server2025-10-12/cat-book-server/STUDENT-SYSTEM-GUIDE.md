# 🎓 学生社团撸猫预约系统使用指南

## 📋 系统更新说明

根据甲方反馈，系统已从"手机号+姓名"认证模式更新为**学生社团直接使用模式**，主要变化：

### ✅ 核心改进
- ❌ **移除**：手机号验证和用户登录系统
- ✅ **新增**：班级选择和学生信息管理
- ✅ **增强**：真实时间段预约数据显示
- ✅ **优化**：5个位置详细预约情况展示
- ✅ **支持**：同一学生可预约多个时间段

## 🚀 新功能详解

### 1. 📅 真实时间段数据
- **余位显示**：时间段卡片显示真实的剩余位置数
- **点击查看详情**：点击时间段进入详细页面
- **5个位置可视化**：清晰展示每个位置的预约状态

### 2. 👥 位置级预约管理
- **位置展示**：每个时间段有5个独立位置
- **学生信息显示**：已预约位置显示学生班级和姓名
- **一键预约**：点击空位置的"+"号即可快速预约
- **预约取消**：已预约位置可直接取消

### 3. 🎒 学生社团模式
- **班级选择**：预设常用班级下拉选项
- **信息缓存**：自动记住学生的班级和姓名
- **多次预约**：同一学生可预约多个时间段
- **查询功能**：通过班级+姓名查询所有预约

## 📱 使用流程

### 预约流程
1. **选择日期** → 在预约页面选择想要的日期
2. **查看时间段** → 系统显示所有时间段和余位信息
3. **点击详情** → 点击感兴趣的时间段查看5个位置详情
4. **选择位置** → 点击空位置的"+"号开始预约
5. **填写信息** → 选择班级、输入姓名和备注
6. **确认预约** → 提交后立即生效

### 查询流程
1. **进入查询** → 点击导航栏"查询预约"
2. **选择班级** → 从下拉菜单选择班级
3. **输入姓名** → 填写学生姓名
4. **查看结果** → 显示该学生的所有预约记录
5. **管理预约** → 可直接取消未过期的预约

## 🛠️ 技术特性

### API对接
- ✅ `/api/available-slots` - 获取时间段列表
- ✅ `/api/available-slots/detail` - 获取位置详情
- ✅ `/api/bookings` - 创建/查询预约
- ✅ `/api/bookings/by-student` - 按学生查询
- ✅ `/api/bookings/{id}` - 取消预约

### 数据模型
```javascript
// 预约数据结构
{
    id: 123,
    date: "2023-12-15",
    time: "14:00",
    studentName: "张三",
    studentClass: "计算机1班", 
    position: 1,
    note: "第一次来",
    status: "CONFIRMED",
    studentIdentity: "计算机1班 - 张三",
    timeSlotDescription: "14:00-14:30"
}
```
